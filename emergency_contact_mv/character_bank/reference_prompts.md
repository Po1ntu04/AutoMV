# Character Reference Prompt Pack

Use `label.json` as the identity source. Generate reference images before any
shot keyframes. Keep all outputs in the same original modern anime film style.

## Global Style

Original modern anime film look, cinematic 2D animation, realistic anime
proportions, detailed painted background, subtle cel shading, restrained acting,
low-saturation color, delicate film grain, quiet psychological drama, 16:9
widescreen composition, no singer performance, no lip-sync, no readable text.

Negative: no celebrity likeness, no real singer likeness, no readable text, no
watermark, no chibi, no idol pose, no glamour styling, no gore, no corpse, no
front-facing singing, no exaggerated melodrama, no drastic style change.

- Reject candidates that make her look like a different woman after changing
